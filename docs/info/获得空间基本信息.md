# 获得空间基本信息

## 获得空间基本信息


> POST http://c6.ysepan.com/nkj/csxx.aspx?cz=dq

要求提供 `Authorization`，例如 `Bearer false;17672942456789000`

可选参数 `Origin` `Referer`，分别例如 `http://false.ysepan.com` `http://false.ysepan.com/`

<details>
<summary>参考请求头</summary>

```http
POST /nkj/csxx.aspx?cz=dq HTTP/1.1
Host: c6.ysepan.com
Content-Length: 0
Authorization: `Bearer false;17672942456789000`
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/146.0.0.0 Safari/537.36
Accept: application/json, text/plain, */*
Content-Type: application/x-www-form-urlencoded
Origin: http://false.ysepan.com
Referer: http://false.ysepan.com/
Accept-Encoding: gzip, deflate
Accept-Language: zh-CN,zh;q=0.9
```
</details>
